# Git 초기화 및 원격 저장소 연결

최초 프로젝트 생성 후 Git에 등록하는 과정입니다.

## 단계별 명령어

### 1. Git 저장소 초기화
```bash
git init
```

### 2. README.md 파일 추가
```bash
git add README.md
```

### 3. 첫 커밋 생성
```bash
git commit -m "first commit"
```

### 4. 기본 브랜치를 main으로 변경
```bash
git branch -M main
```

### 5. 원격 저장소 추가
```bash
git remote add origin https://github.com/trance81/smart-monthly-budget-planner.git
```

### 6. 원격 저장소로 푸시
```bash
git push -u origin main
```

## 참고사항

- 위 명령어들은 순서대로 실행해야 합니다.
- `git push -u origin main` 명령어는 원격 저장소에 main 브랜치를 푸시하고, 이후 `git push`만으로도 푸시할 수 있도록 upstream을 설정합니다.
